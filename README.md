# IMAP Proxy (SSH Tunnel Endpoint)

## Назначение
Принимает IMAP-трафик (TCP 993) на `localhost:10993` хоста сервера и проксирует его
в Docker-сеть к почтовому контейнеру. Используется как целевая точка для
SSH Remote Forwarding, чтобы обойти блокировку порта 993 на фаерволе сервера.

## Схема
```
Thunderbird (удалённое устройство)
    │
    │ IMAP over TLS на порт 993
    ▼
ssh -R 10993:localhost:993  ← выполняется с ЛОКАЛЬНОЙ Ubuntu-машины
    │
    │ SSH (исходящий порт 22 — открыт)
    ▼
Сервер:127.0.0.1:10993 → HAProxy → mail-container:993
```

## Использование на клиенте (локальная Ubuntu-машина)
```bash
ssh -R 10993:localhost:993 user@server_ip
```

Для автоматического поддержания туннеля рекомендуется `autossh`:
```bash
autossh -M 0 -fNR 10993:localhost:993 user@server_ip
```

Флаги:
- `-f` — перейти в фон после успешного подключения
- `-N` — не выполнять удалённые команды (только проброс портов)
- `-R` — remote port forwarding

## Потребители сервиса
Сервис используется `support` (Zammad) для подключения к IMAP-почте.
